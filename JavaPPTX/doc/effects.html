The edu.stanford.cs.pptx package implements the following effects:

Appear
This entrance effect causes the shape to appear in response to the trigger event, which defaults to /onClick. The legal options are:
/onClick Effect takes place after a mouse click
/afterPrev Effect takes place after the previous one
/withPrev Effect takes place with the previous one
/delay:time Effect is delayed by time seconds

ChangeFillColor
This emphasis effect change the fill color of the shape gradually over the duration of the effect. The legal options are:
/onClick Effect takes place after a mouse click
/afterPrev Effect takes place after the previous one
/withPrev Effect takes place with the previous one
/delay:time Effect is delayed by time seconds
/duration:time Effect lasts time seconds
/color:color Specifies the target color
